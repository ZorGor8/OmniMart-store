OmniMart: A Type-Safe E-commerce Platform
OmniMart is a modern, high-performance single-page application (SPA) built with React and TypeScript. The project focuses on clean architecture, scalable state management, and a seamless user experience.

## Technical Highlights 
Advanced State Management: Implemented a sophisticated shopping cart logic using React Hooks, handling complex operations like item persistence, quantity increments, and real-time total calculations.
[Сложная логика корзины с использованием React Hooks: управление количеством и расчет итогов в реальном времени.]

Scalable Type System: Leveraged TypeScript interfaces and inheritance to create a robust data model, ensuring type safety across all components and props.
[Масштабируемая система типов на TypeScript для надежности кода.]

Custom Responsive Grid: Developed a proprietary CSS Grid system to ensure a fluid and adaptive product catalog across all device sizes (Desktop, Tablet, Mobile).
[Собственная адаптивная сетка на CSS Grid для всех типов устройств.]

Dynamic Routing & Filtering: Integrated React Router v6 for efficient client-side navigation and category-based product filtering.
[Динамический роутинг и фильтрация товаров по категориям.]
